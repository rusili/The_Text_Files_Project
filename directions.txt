Git status -> shows current status of local repository

Git log -> shows history of your commits

Git add (filename) -> adds file/files to staging area to be committed

Git commit -m “(commit message)” -> commits files to local repository. Makes a snapshot of the current state of your directory.

Git remote add (name) git@github.com/(username)/(repositoryname) -> connects your local repository to an online repository to push to.

Git push -u (name) (branch) -> pushes local commits to your online repository.
